# SPARK: A Student-Centered Framework for Teaching NLP in the Age of Large Language Models

This repository accompanies the **SPARK** framework (**Scaffolded, Participatory, Active, Reflective Knowledge-building**), a student-centered pedagogical framework for teaching Natural Language Processing (NLP) in the era of large language models (LLMs).

Developed and studied over four years in the **Bachelor of Computer Science at the University of Luxembourg**, SPARK is designed to help undergraduate students learn **with and about AI systems** through scaffolded, reflective, and research-oriented learning experiences. Rather than treating AI as a shortcut, SPARK positions it as both an **object of inquiry** and a **collaborative partner**.

## Overview

SPARK is grounded in:
- Social constructivism
- Active learning
- Project-based pedagogy
- Psychological safety
- Assessment alignment

The framework combines:
- Hands-on implementation
- Peer-led learning
- Student presentations
- Low-stakes gamification
- Research-inspired final projects

Students progressively move from **glass-box NLP methods** (e.g., preprocessing, TF-IDF, n-grams, classical machine learning) to **black-box and AI-rich systems** such as:
- Transformer-based models
- Fine-tuning
- Retrieval-Augmented Generation (RAG)
- Multi-agent systems
- Explainable AI applications

## Example Student Projects

Below are selected student projects developed within the course:

### 1. Agentic Socratic NLP Tutor
A modular agentic tutoring system combining question generation, response analysis, and dialogue management to produce adaptive Socratic follow-up questions grounded in student responses.  
**Repository:**  
https://github.com/RedPill47/Agentic_Socratic_NLP_Tutor-final

### 2. Explained AI-Text Detection Using Curvature-Based Signals
An AI-generated text detection system based on curvature-related statistical features, paired with an explanation module that visualizes confidence patterns and supports failure analysis.  
**Repository:**  
https://github.com/TenderChasm/Notai.ai

### 3. Book Recommendation Chatbot Using RAG
A retrieval-augmented generation chatbot that retrieves relevant book information from an indexed collection and conditions responses on grounded evidence.  
**Demo:**  
https://drive.google.com/file/d/1kyxr2nF-k0n8jdb8afXdE98445PFeOo-/view?usp=sharing  
**Repository:**  
https://github.com/BiancaGL2104/rag-book-recommender

### 4. Multi-Agent Résumé and Cover Letter Assistant
A multi-agent workflow for information extraction, skill matching, document planning, and controlled generation of résumés and cover letters.  
**Demo:**  
https://drive.google.com/file/d/19dvWKpDye3bZkp2uljUpis6uL_igrJoe/view?usp=sharing

## Key Takeaways from the Study

Across multiple course iterations, the framework showed that undergraduate students can meaningfully engage with advanced AI systems when learning is scaffolded to preserve:
- Explanation
- Evaluation
- Reflection
- Learner agency

Students demonstrated growth in:
- Technical competence in foundational and modern NLP
- Critical reasoning
- Research readiness
- Communication
- Human–AI collaboration skills

One student group summarized their experience as follows:

> “This was the only course this semester where we attended every single session, because we always felt we were learning something meaningful.”
